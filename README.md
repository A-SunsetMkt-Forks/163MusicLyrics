### How To Use

[👉163MusicLyrics Wiki](https://github.com/jitwxs/163MusicLyrics/wiki)

- [x] 支持网易云、QQ音乐两家音乐提供商
- [x] 支持输入歌曲 ID、输入专辑 ID、完整链接的方式进行查询
- [x] 支持批量查询
- [x] 支持多种歌词原文和译文的组织方式
- [x] 支持提取（部分）歌曲试听链接
- [x] 支持多种保存命名规则、文件编码、输出格式
